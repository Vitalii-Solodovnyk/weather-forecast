# Weather App

A simple and stylish web application to check the current weather in any city around the world. Built with vanilla JavaScript and the OpenWeatherMap API.

## Features

* **City Search:** Enter the name of any city (e.g., "London", "New York", "Tokyo").
* **Real-time Data:** Displays the current temperature in Celsius.
* **Error Handling:** Displays a user-friendly error message directly on the screen if the city is not found or the input is invalid.
* **Responsive UI:** Clean, centered layout using CSS Flexbox with a dynamic background.

## Technologies Used

* **HTML5** – Semantic structure.
* **CSS3** – Styling, Flexbox for layout, and responsive design.
* **JavaScript (ES6+)** – DOM manipulation, event handling (`click`, `input`), and asynchronous logic.
* **Fetch API** – For making HTTP requests to get weather data.
* **OpenWeatherMap API** – Source of weather data.
